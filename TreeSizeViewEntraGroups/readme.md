My aim was to visually represent nested groups in Entra ID.

V3 displays a visualisation in the terminal; 
see an example below.
It primarily works by sorting, arranging, and then assigning a ‘level’ to each group.

V4 generates a JSON file;
Here, a group number is added for each tree,
and the child group IDs are also included.
Finally, the complete nested structure is saved in a JSON file.

You can upload the JSON file into ‘nested-groups-viewer.html’ 

This provides a different type of visualisation and allows you to search for groups.

In V5 (future), I will look into optimising speed.
